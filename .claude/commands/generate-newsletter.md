# ニュースレター生成

以下の情報に基づいてニュースレターを生成してください。

## 入力データ
$ARGUMENTS

## 指示

1. **ニュース収集**: WebSearchを使用して、上記の関心領域（interests）に関連する直近1週間の重要なニュースを検索してください。各関心領域について2-3件のニュースを収集します。

2. **ニュース選定**: 収集したニュースから、ビジネスインパクトが大きく新規性のあるものを5-7件選定してください。

3. **コンテンツ作成**: 以下の構成で作成：
   - エグゼクティブサマリー（100-150字）：今週の重要トピックを要約
   - 注目ニュース（各80-120字）：タイトル、URL、ソース、概要
   - 分析・考察（150-200字）：トレンドとビジネスへの示唆

4. **出力**: 以下のJSON形式で `output/newsletter.json` に保存してください：

```json
{
  "title": "ニュースレタータイトル",
  "date": "YYYY年MM月DD日",
  "summary": "エグゼクティブサマリー",
  "articles": [
    {
      "title": "記事タイトル",
      "url": "https://...",
      "source": "ソース名",
      "date": "YYYY/MM/DD",
      "description": "記事の概要と意義"
    }
  ],
  "analysis": "分析・考察"
}
```

## 注意事項
- 客観的かつ専門的なトーンを維持
- ソースURLは実在するものを使用
- 日本語で出力
